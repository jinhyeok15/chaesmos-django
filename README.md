* 이채원 [Team Leader] - 기획 및 UI, UX 디자인

* 이진혁 [Dev Leader]- 백엔드 개발 및 Frontend 반응형 페이지 작업. 코드 리팩토링 진행. 코드 배포
* 정푸름 [Front-end] - 고민 해결하기, 내 메일함 페이지 프론트엔드 개발
